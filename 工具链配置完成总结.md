## 🔧 为什么选择 GNU 工具链

### 项目需求

1. **跨平台一致性**
   - GNU 工具链在 Windows、Linux、macOS 上行为一致
   - 适合 Rust 生态系统

2. **更小的二进制文件**
   - 生成的可执行文件更小
   - 减少分发大小

3. **更好的 POSIX 兼容性**
   - 与 Unix-like 系统兼容
   - 便于代码移植

4. **免费开源**
   - 完全开源，无许可限制
   - 社区支持良好

### GNU vs MSVC 对比

| 特性 | GNU (MinGW) | MSVC |
|------|-------------|------|
| 编译器 | GCC | Microsoft C++ |
| 链接器 | ld | link.exe |
| 运行时 | MinGW-w64 | MSVCRT |
| 文件大小 | 较小 | 较大 |
| 启动速度 | 较快 | 较慢 |
| 调试支持 | GDB | Visual Studio |
| 跨平台 | 优秀 | 一般 |
