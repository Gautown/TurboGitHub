# TurboGitHub 快速发布指南

## 🚀 5 分钟快速发布

### 方法 1：一键打包（最简单）

```bash
# 1. 构建并打包
.\打包发布.bat

# 2. 等待完成
# 输出：release/TurboGitHub-{version}-portable.7z
#      release/TurboGitHub-{version}-portable.zip
```

### 方法 2：手动打包

```bash
# 1. 构建 Release
cargo build --release

# 2. 创建发布目录
mkdir release\TurboGitHub-0.0.1

# 3. 复制文件
copy target\release\turbogithub-gui.exe release\TurboGitHub-0.0.1\
copy config.toml release\TurboGitHub-0.0.1\
copy 启动 TurboGitHub.bat release\TurboGitHub-0.0.1\
xcopy /E /I assets release\TurboGitHub-0.0.1\assets

# 4. 压缩
cd release
7z a -t7z -mmt=on TurboGitHub-0.0.1-portable.7z TurboGitHub-0.0.1
7z a -tzip -mmt=on TurboGitHub-0.0.1-portable.zip TurboGitHub-0.0.1
```

## 📦 创建安装包（可选）

### 使用 Inno Setup

```bash
# 1. 安装 Inno Setup
# 下载：https://jrsoftware.org/isdl.php

# 2. 编译安装包
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss

# 3. 输出：installer_output/TurboGitHub-0.0.1-Setup.exe
```

## 🏷️ 发布到 GitHub

### 1. 提交代码

```bash
git add -A
git commit -m "release: v0.0.1 首次发布"
git push TurboGitHub main
```

### 2. 打标签

```bash
git tag -a v0.0.1 -m "TurboGitHub v0.0.1"
git push TurboGitHub v0.0.1
```

### 3. 创建 Release

1. 访问 https://github.com/Gautown/TurboGitHub/releases/new
2. 选择标签 `v0.0.1`
3. 填写发布说明
4. 上传文件：
   - TurboGitHub-0.0.1-portable.7z
   - TurboGitHub-0.0.1-portable.zip
   - TurboGitHub-0.0.1-Setup.exe (可选)
5. 点击 "Publish release"

## ⚡ 自动化发布（推荐）

### 使用 GitHub Actions

```bash
# 1. 打标签并推送
git tag -a v0.0.1 -m "TurboGitHub v0.0.1"
git push TurboGitHub v0.0.1

# 2. GitHub Actions 会自动：
#    - 构建 Release 版本
#    - 创建压缩包
#    - 生成 SHA256 校验和
#    - 创建 GitHub Release
#    - 上传所有文件
```

## 📊 文件大小优化

### Cargo.toml 已优化

```toml
[profile.release]
opt-level = 3        # 最大性能优化
lto = "thin"         # 轻量级 LTO
codegen-units = 4    # 多个代码单元
panic = "abort"      # 使用 abort
strip = true         # 剥离调试符号
```

### 进一步压缩（可选）

```bash
# 使用 UPX 压缩可执行文件
cargo install --force cargo-binstall
cargo binstall upx

upx --best --lzma target\release\turbogithub-gui.exe
# 可减少 50-70% 体积
```

## 🎯 推荐发布流程

### 首次发布

1. **本地测试**
   ```bash
   cargo build --release
   .\target\release\turbogithub-gui.exe
   ```

2. **打包**
   ```bash
   .\打包发布.bat
   ```

3. **测试安装包**
   ```bash
   # 在干净系统上测试
   installer_output\TurboGitHub-0.0.1-Setup.exe
   ```

4. **发布**
   ```bash
   git add -A
   git commit -m "release: v0.0.1"
   git tag -a v0.0.1 -m "首次发布"
   git push TurboGitHub main
   git push TurboGitHub v0.0.1
   ```

5. **创建 GitHub Release**
   - 上传所有文件
   - 填写发布说明
   - 发布

### 后续发布

1. 更新版本号
2. 更新 CHANGELOG.md
3. 打标签推送
4. GitHub Actions 自动发布

## 📞 故障排除

### Q: 构建失败
```bash
cargo clean
cargo update
cargo build --release
```

### Q: 打包失败
检查文件路径，确保 `target\release\turbogithub-gui.exe` 存在

### Q: GitHub Actions 失败
检查 workflows 配置和标签格式（v0.0.1）

### Q: 安装包创建失败
检查 Inno Setup 版本和 installer.iss 语法
